# 💻 Cartão de Perfil

Esse projeto foi desenvolvido para praticar HTML e CSS, criando um card de perfil simples, organizado e visualmente agradável.

---

## 🛠️ Tecnologias utilizadas
- HTML
- CSS

---

## 🎯 Objetivo
Treinar a estruturação de páginas e melhorar o uso de CSS para estilização e posicionamento de elementos.

---

## 🌐 Visualizar projeto
https://liadaine.github.io/card-de-perfil/

---

## 📚 O que foi praticado
- Estrutura básica em HTML  
- Estilização com CSS  
- Alinhamento e organização de elementos  
- Criação de um layout simples e funcional
